## Descrição
Este repositório foi criado para organizar os materiais do curso introdutório de Python para Ciência de Dados, desenvolvido como atividade extensionista da disciplina de Ciência de Dados da Universidade de Fortaleza (UNIFOR).

## Objetivo
Oferecer uma formação introdutória em Python aplicada à Ciência de Dados, capacitando iniciantes a compreender conceitos básicos de programação, manipulação, análise e visualização de dados.

## Público-alvo
Estudantes do ensino médio, técnico ou superior, profissionais em transição de carreira e demais interessados em iniciar na programação com foco em Ciência de Dados.

## Formato do curso
Curso introdutório, com aulas expositivas e práticas, utilizando exemplos simples, notebooks comentados e exercícios.

## Carga/estrutura inicial
Curso dividido em 4 módulos e 1 projeto final, organizado de forma progressiva, partindo dos fundamentos de Python até a análise exploratória de dados.

## Escopo fechado do curso
O curso “Introdução ao Python para Ciência de Dados” será voltado ao público iniciante, especialmente estudantes e membros da comunidade externa interessados em adquirir conhecimentos básicos de programação e análise de dados. O curso terá caráter introdutório e prático, abordando desde os fundamentos da linguagem Python até noções iniciais de manipulação, visualização e análise exploratória de dados (EDA). A proposta é oferecer uma formação acessível, aplicada e contextualizada, permitindo que os participantes desenvolvam habilidades básicas para utilização de Python em atividades acadêmicas e profissionais. Não serão abordados conteúdos avançados, como machine learning, banco de dados ou desenvolvimento web, mantendo o foco na formação introdutória em Ciência de Dados.

## Conteúdo planejado
- Módulo 1: Fundamentos de Python
- Módulo 2: Manipulação de Dados
- Módulo 3: Visualização de Dados
- Módulo 4: Análise Exploratória de Dados (EDA)
- Projeto final: aplicação prática dos conteúdos

## Ementa detalhada até EDA

### Módulo 1 – Fundamentos de Python
Introdução à linguagem Python, ambiente de desenvolvimento, sintaxe básica, variáveis, tipos de dados, operadores, entrada e saída de dados, estruturas condicionais, estruturas de repetição, listas, tuplas, dicionários e boas práticas de programação.

### Módulo 2 – Manipulação de Dados
Introdução à biblioteca Pandas, leitura de arquivos CSV, estrutura de tabelas, seleção de colunas e linhas, filtragem, criação de colunas, tratamento básico de valores ausentes e estatísticas descritivas simples.

### Módulo 3 – Visualização de Dados
Introdução à visualização de dados, uso de bibliotecas como Matplotlib e Seaborn, criação de gráficos de barras, linhas, histogramas e dispersão, além de noções básicas de interpretação visual.

### Módulo 4 – Análise Exploratória de Dados (EDA)
Exploração inicial de bases de dados, identificação de padrões, tendências e inconsistências, formulação de perguntas a partir dos dados, interpretação de resultados e comunicação de insights iniciais.

### Projeto final
Aplicação integrada dos conteúdos em uma pequena análise com dataset simples, envolvendo leitura, manipulação, visualização e interpretação de dados.

## Pré-requisitos definidos
O curso não exigirá conhecimento prévio em programação, por se tratar de uma formação introdutória. Como pré-requisitos, espera-se apenas que os participantes possuam noções básicas de informática, saibam utilizar computador, navegador de internet e manipular arquivos e pastas. Também será necessário ter acesso à internet e a um ambiente para execução dos códigos, como Google Colab, Jupyter Notebook ou outra plataforma similar.

### Notebook
O notebook do Módulo 1 pode ser executado de duas formas:
- pelo Google Colab, fazendo upload do arquivo `.ipynb`;
- localmente, utilizando Jupyter Notebook.
